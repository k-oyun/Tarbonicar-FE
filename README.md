# 🧑🏻‍💻 타보니까
#### 현대자동차 커뮤니티 플랫폼
#### 현대자동차 고객들의 실질적인 경험을 바탕으로 차종별, 차량별, 연식별로 정보를 손쉽게 탐색할 수 있는 사용자 맞춤형 웹 서비스입니다.
<br />


# 🧾 기능
### 필터링 시스템
##### - 커스텀 셀렉트박스 / 체크박스를 이용하여 차종, 차량, 연식 선택
### 게시글 
##### - 제목, 본문, 이미지, 차량정보, 카테고리를 입력하여 게시글 작성
##### - 차종, 차량, 연식, 카테고리별 필터링
### 댓글 및 좋아요 기능
##### - 댓글 수정 및 삭제
##### - 좋아요 등록 및 취소
### 마이페이지
#### 게시글 관리
##### - 내가 작성한 글 목록 조회
##### - 최신순, 오래된순, 좋아요순, 조회수순으로 정렬 가능
#### 회원정보 수정
##### - 닉네임, 비밀번호, 프로필 이미지 변경 및 회원 탈퇴
